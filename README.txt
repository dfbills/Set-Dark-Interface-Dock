# set-dark-interface-dock
------------------------------
Created by David F. Bills on 12.04.14


# About
----------

set-dark-interface-dock sets the MacOSX Yosemite Dock to the dark interface style without changing the entire UI to dark mode.  The setting are cleared on reboot.

Included in this package is an installer and a LaunchAgent to reapply the settings on login.  

Tested with 10.10.1


# To Install
----------
Run install.sh and specify installation directory.

> install.sh ~/Application
